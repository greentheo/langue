# Changelog - v0.3.0 Flashcard Libraries Update

## Overview

Version 0.3.0 introduces a comprehensive flashcard library system that organizes vocabulary by language proficiency level (A1-C2), providing better control over the content presented to language learners and enabling more structured learning paths.

## Major Features

### Flashcard Library System
- Added structured vocabulary libraries organized by language and CEFR level (A1-C2)
- Implemented vocabulary library generator using LLMs
- Created CLI tool for generating level-appropriate vocabulary
- Added library management system for efficient vocabulary access
- Designed flexible JSON format for storing word data with multiple translations

### Language Level Integration
- Added language level selection in the main menu
- Enhanced user profiles to track language level preferences
- Updated all activities to use level-appropriate content
- Improved display of level information throughout the UI
- Added automatic level mapping from difficulty settings

### Multiple Translation Support
- Enhanced flashcard system to handle words with multiple valid translations
- Updated evaluation system to recognize all valid translations as correct
- Improved display to show all accepted translations to users
- Modified LLM prompt to be more generous with scoring when any valid translation is provided
- Enhanced fallback evaluation to check against all translations

### Weighted Selection Algorithm
- Implemented intelligent word selection that prioritizes challenging words
- Created weighted algorithm that gives higher weight to words with lower scores
- Added special handling for new words to encourage exploration
- Implemented caching for performance optimization

## Improvements

### UI Enhancements
- Updated main menu to include "Change Language or Level" option
- Enhanced activity display to show current language and level
- Improved flashcard display with better formatting for multiple translations
- Added detailed level information in user profile summary

### Learning Experience
- More generous scoring system that rewards partial understanding
- Clearer feedback that shows all acceptable translations
- Prioritization of difficult words for more efficient learning
- Progress tracking against level-appropriate vocabulary

### Code Organization
- Created modular library system with clear separation of concerns
- Added comprehensive error handling for library operations
- Implemented caching for efficient library access
- Enhanced testing framework for vocabulary libraries

## Bug Fixes
- Fixed handling of words with multiple translations
- Resolved evaluation issues with alternate spellings
- Fixed error handling in library file operations
- Improved fallback behavior when libraries aren't available
- Added offline mode for library generation without model access

## Developer Improvements
- Added detailed documentation of library system
- Created test suite for multiple translation handling
- Added utility methods for working with vocabulary libraries
- Improved code organization with clear module structure

## Notes
- To generate vocabulary libraries, use the new `langue library` command
- For offline use without Ollama or internet, use `langue library --offline`
- Libraries can be created directly with `python -m langue.tools.library_generator`
- Existing flashcards will continue to work alongside the library system
- The system automatically falls back to model-generated content when libraries aren't available
